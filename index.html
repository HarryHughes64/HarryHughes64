HTML Website Basics

Basic, key HTML elements you need to know when starting your HTML website build, along with some more specific attributes. This Cheatsheet also contains semantic examples too.

Tag Name Description
<!DOCTYPE html> Document Type This is the first line of code used to inform the type of document it is - e.g. html
<html lang=""> Language This is the second line of code, used to inform what specific language is expected (en = English).
<head> Metadata The head element should contain any metadata, such as the title, character set and more.
<header> Header The header element is semantic and used to contain logo's, nav bars or head (h1...) elements
Class="" Class A class can be used inside any element, typically used to group and seperate elements - a class can be used by multiple elements. The key is to name the class describing what the element does, rather than how it looks.
id="" ID An ID is used to specify an individual html elements. Remeber to name the ID describing what the element does, rather than how it looks.
<table> Table Starts a table element
<tr> Table Row The table row container
<th> Table Row The table row headers
<td> Table Data Contains the contents of the table cell
colspan="" Column Span Used as an attribute, this sets the number of column the cell spans over
rowspan="" Row Span Used as an attribute, this sets the number of rows the cell spans over
